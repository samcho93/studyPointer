# StudyPointer - C언어 포인터 완전정복

64비트 시스템 기반 메모리 시각화로 배우는 C 포인터 웹 학습 프로그램입니다.
C++ 포인터를 위한 예비 과정으로, 총 **20개 강좌**와 **102장 슬라이드**를 제공합니다.

---

## 주요 기능

- **20개 단계별 강좌** — 메모리 구조부터 함수 포인터까지 난이도순 구성
- **메모리 시각화** — 매 강좌마다 스택/힙/데이터/코드 영역의 메모리 맵 제공
- **C 구문 강조** — 토크나이저 기반 신택스 하이라이팅
- **PPT 스타일 슬라이드** — 전체화면 프레젠테이션 모드 (요약 + 코드 + 메모리맵 + 핵심포인트)
- **학습 진도 저장** — localStorage 기반 자동 저장
- **네트워크 공유** — 동일 네트워크 PC에서 브라우저로 접속 가능

## 설치 및 실행

### 요구 사항

- **Python 3.x** (내장 HTTP 서버 사용)
- 최신 웹 브라우저 (Chrome, Edge, Firefox 등)

### 1. 저장소 클론

```bash
git clone https://github.com/samcho93/studyPointer.git
cd studyPointer
```

### 2. 로컬 서버 실행

```bash
python -m http.server 8080 --bind 0.0.0.0 --directory .
```

### 3. 브라우저에서 접속

```
http://localhost:8080
```

## 네트워크 공유 (동일 네트워크 PC 접속)

같은 네트워크에 있는 다른 PC에서 강의를 볼 수 있습니다.

### 1단계: 서버 PC의 IP 확인

```bash
# Windows
ipconfig

# Mac/Linux
ifconfig
```

IPv4 주소를 확인합니다 (예: `192.168.0.10`).

### 2단계: 방화벽 포트 허용 (Windows)

관리자 권한 PowerShell에서 실행:

```powershell
netsh advfirewall firewall add rule name="StudyPointer" dir=in action=allow protocol=tcp localport=8080
```

### 3단계: 다른 PC에서 접속

브라우저에서 서버 PC의 IP로 접속:

```
http://192.168.0.10:8080
```

## 사용법

### 강좌 학습

1. 좌측 사이드바에서 강좌를 선택하거나, **첫 번째 강좌 시작하기** 클릭
2. 각 강좌는 **개념 설명 → 예제 코드 → 메모리 맵 → 핵심 포인트** 순서로 구성
3. 상단/하단의 이전/다음 버튼 또는 **← →** 키보드로 이동

### 슬라이드 모드

1. 환영 화면의 **슬라이드로 전체 보기** 버튼 클릭
2. 또는 강좌 화면에서 **▶ 슬라이드** 버튼으로 해당 강좌부터 시작
3. **← →** 키보드, **Home/End** 키, **ESC**로 닫기

### PPT 파일 생성 (선택)

```bash
pip install python-pptx
python generate_ppt.py
```

`C언어_포인터_완전정복.pptx` 파일이 생성됩니다.

## 기술 스택

- **프론트엔드**: Vanilla HTML / CSS / JavaScript (프레임워크 없음)
- **서버**: Python 내장 HTTP 서버
- **구문 강조**: 토크나이저 기반 자체 구현 (정규식 간섭 방지)
- **메모리 시각화**: HTML 테이블 + CSS 컬러코딩
- **슬라이드**: DOM 파싱 기반 자동 요약 생성
